# 🎓 EduBot AI — Asisten Belajar Cerdas

**Hacktiv8 Final Project** — AI Productivity and AI API Integration for Developers

---

## 📌 Deskripsi

EduBot AI adalah chatbot berbasis AI yang dirancang untuk membantu pengguna belajar berbagai mata pelajaran secara interaktif. Chatbot menggunakan model Claude (Anthropic) untuk memproses bahasa alami dan memberikan penjelasan yang mudah dipahami.

**Use Case:** Education Bot — memberikan edukasi mengenai Python, Matematika, Sains, dan pengetahuan umum lainnya.

---

## ✨ Fitur

- 🌐 **4 Mode Mata Pelajaran** — Umum, Matematika, Coding, Sains
- 💬 **Riwayat Percakapan** — Bot mengingat konteks chat sebelumnya
- ⚡ **Quick Questions** — Pertanyaan siap pakai sekali tap
- 🎨 **UI Modern** — Dark theme yang clean dan responsif
- 📖 **Panduan API Key** — Panduan lengkap cara mendapatkan API Key built-in di dalam app
- 📱 **Responsif** — Bisa dipakai di HP maupun PC

## 🚀 Cara Menjalankan

### Online (GitHub Pages)
Akses langsung di: `https://bayusantosowijaya-arch.github.io/Edubot-ai`

### Lokal
1. Download file `edubot-ai.html`
2. Buka dengan browser (Chrome/Firefox/Edge)
3. Masukkan Anthropic API Key saat diminta

---

## 🔑 Cara Mendapatkan API Key

1. Daftar di [console.anthropic.com](https://console.anthropic.com)
2. Masuk ke menu **API Keys**
3. Klik **Create Key**
4. Copy key dan paste ke aplikasi
