Cara Membuat API Key

Ikuti langkah berikut untuk mendapatkan API Key Anthropic secara gratis (ada free credit $5):

⚠️ Jangan pernah share API Key kamu ke siapapun. Simpan seperti password!
1

Buka Anthropic Console

Kunjungi website resmi Anthropic dan buat akun gratis.

https://console.anthropic.com
2

Daftar / Login

Klik Sign Up dan daftar pakai email Google atau email biasa. Verifikasi email kamu.

3

Masuk ke API Keys

Setelah login, klik menu API Keys di sidebar kiri, atau langsung ke:

https://console.anthropic.com/settings/keys
4

Create New Key

Klik tombol + Create Key, beri nama bebas (misal: "EduBot"), lalu klik Create Key.

5

Copy API Key

Key akan muncul sekali saja — langsung copy dan simpan! Formatnya seperti ini:

sk-ant-api03-xxxxxxxxxxxx...
6

Paste di Tab "Masukkan Key"

Kembali ke tab Masukkan Key di atas, paste key kamu, lalu klik Aktifkan.
